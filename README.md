This repository contains the code for the simulations in "Communication cost of consensus for nodes with limited memory" (Fanti, Holden, Peres, Ranade).

## Dependencies:
- python3
- numpy
- matplotlib
- collections
- cProfile

## Figure 4
1) Uncomment the configurations in section 'Figure 4' of config.py
2) Run `python main.py`
